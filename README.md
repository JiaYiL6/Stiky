# Stiky - Windows 桌面便签 & 文件中转站

模仿 Windows 自带便笺，增加置顶、透明度调节和文件中转站功能。

## 功能

### 便签
- **富文本编辑**：支持加粗、斜体、下划线、删除线、无序列表、有序列表、待办（可勾选）
- **图片支持**：支持粘贴图片，拖拽边缘调整尺寸（Shift 保持比例），双击全屏预览
- **多彩主题**：8 种颜色可选，每张便签独立设置
- **置顶**：📌 一键将便签窗口保持在其他窗口之上
- **透明度**：🔆 滑块调节 10%-100%，鼠标悬停时自动恢复不透明
- **字号**：设置面板统一调整全局默认字号
- **自动隐藏**：鼠标移出窗口后标题栏和底部工具栏自动淡出（0.6 秒），保持干净简洁
- **空白清理**：关闭空白便签自动删除，不留下空便签

### 文件中转站
- **直接引用原文件**：拖入文件不产生副本，直接引用原始路径，零磁盘占用
- **双击打开编辑**：双击文件用系统默认程序打开，编辑直接保存回原文件
- **拖出使用**：从中转站直接拖拽文件到微信、邮件、文件夹等任意位置
- **Windows 缩略图**：图片显示真实预览，文档显示系统关联图标
- **独立于便签**：每张便签拥有独立的中转站，互不干扰
- **hover 展开**：鼠标移到窗口右边缘自动展开侧边栏，移开 0.6 秒后收起
- **图钉固定**：📌 点击固定侧边栏保持展开

### 系统功能
- **系统托盘**：启动后仅驻留托盘，不打开窗口；双击托盘图标或右键菜单新建便签
- **全局快捷键**：`Alt+N` 新建便签，`Alt+M` 打开便签管理器
- **便签管理器**：集中查看所有便签，搜索、批量删除、单独修改颜色，点击重新打开已关闭的便签
- **统一设置**：便签默认值、缩略图开关、任务栏显示、开机启动

## 安装

### 方式一：安装包（推荐）

下载 [Stiky Setup 1.0.6.exe](https://github.com/JiaYiL6/Stiky/releases) 运行安装。

### 方式二：源码运行

```bash
# 克隆仓库
git clone https://github.com/JiaYiL6/Stiky.git
cd Stiky

# 安装依赖
npm install

# 启动
npm start
```
